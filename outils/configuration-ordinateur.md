## Installs nécessaires

Ouvrir un terminal et éxécuter les commandes suivantes:

- php -v
- composer -v
- mariadb -v
- nodejs -v
- npm -v
- yarn -v
- gulp -v
- git config --global core.excludesFile
- gh version

Toutes les commandes devraient retourner un résultat.
